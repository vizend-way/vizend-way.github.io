REST Support

REST API 를 호출하기위한 WebFlux 기반의 클라이언트 모듈을 제공합니다. 자체로 프로퍼티를 가지고 동작하기 때문에 다음과 같이 @Bean 초기화 후에 사용할 수 있습니다. 아래는 이를 설정하는 예시입니다.

REST API 클라이언트 프로퍼티 설정은 아래와 같이 작성합니다. 필요할 경우 추가 설정을 적용할 수 있습니다.

@Data
@ConfigurationProperties(prefix = "vizend.foo.client")
public class FooClientProperties {
    //
    private RestRequesterProperties rest;

    public CheckpointClientProperties() {
        //
        rest = RestRequesterProperties.builder()
                .baseUrl("http://foo:8080")
                .maxMemorySize(1024 * 1024) // 1 MB
                .build();
    }
}

RestRequesterProperties에서 설정할 수 있는 항목 및 기본 설정은 아래와 같습니다.

base-url: http://200.x.x.10:8080 # REST API 제공 URL
loopback: false                  # 실제 호출하지 않고 빈 값을 리턴할지 여부로 개발용으로 사용
base-method: POST                # 기본 HTTP 요청 메소드
retry: 3                         # 실패시 재시도 회수
request-timeout-seconds: 60      # 요청 타임아웃 (초)
max-memory-size: 52428800        # 기본 50 MB로 응답 사이즈 제한

위 설정에서 사용하는 application.yml 설정 부분은 다음과 같습니다.

vizend:
  foo:
    client:
      rest:
        base-url: http://200.x.x.10:8080
        request-timeout-seconds: 600

마지막으로 위 설정을 사용해 호출하는 부분은 아래와 같이 Blocking 호출, Nun-blocking 호출로 필요한 경우에 따라 다르게 설정할 수 있습니다.

Blocking 호출​

Blocking 호출은 요청을 보내고 응답이 오기까지 블럭되는 방식입니다. 응답 유형으로 받아 처리할 타입을 지정할 수 있고 일반적인 경우 Body의 문자열을 반환합니다.

@Slf4j
@Component
@RequiredArgsConstructor
public class FooRestClient {
    //
    private final FooClientProperties properties;

    private RestRequester rest;

    @PostConstruct
    public void initialize() {
        //
        this.rest = new RestRequester(this.properties.getRest());
    }

    public String registerFoo(RegisterFooRequest request) {
        //
        String path = UriComponentsBuilder
                .fromUriString("/restful/foos")
                .toUriString();

        String response = rest.post(path, request);

        if (response == null) {
            throw new IllegalArgumentException(
                    "Cannot register foo, fooRequest = " + request);
        }

        return response;
    }
    
    public List<Foo> findFoos() {
        //
        String path = UriComponentsBuilder
                .fromUriString("/restful/foos")
                .toUriString();

        List<Foo> response = rest.getList(path, Foo.class);

        if (response == null) {
            throw new IllegalArgumentException("Cannot find foos");
        }

        return response;
    }
    
    public Foo findFoo(String id) {
        //
        String path = UriComponentsBuilder
                .fromUriString("/restful/foos/{id}")
                .buildAndExpand(id)
                .toUriString();

        Foo response = rest.get(path, Foo.class);

        if (response == null) {
            throw new IllegalArgumentException("Cannot find foo with id = " + id);
        }

        return response;
    }   
}

💡 요청으로 객체를 JSON으로, 응답 타입으로 Generic 유형을 받고 싶을 경우가 있습니다. 이 경우 다음과 같은 조합을 이용해 처리할 수 있습니다.

FindUserQuery query = new FindUserQuery(userId);
User<Customer> response = this.rest.post(
                "/feature/user/find-user-query", query,
                new TypeReference<User<Customer>>() {});

Non-blocking 호출​

Non-blocking 호출은 응답으로 Mono 유형이 반환됩니다. 넌블럭 호출을 이용하고자 할 때 사용합니다.

@Slf4j
@Component
@RequiredArgsConstructor
public class FooRestClient {
    //
    private final FooClientProperties properties;

    private AsyncReqestRequester rest;

    @PostConstruct
    public void initialize() {
        //
        this.rest = new RestRequester(this.properties.getRest());
    }

    public String registerFoo(RegisterFooRequest request) {
        //
        String path = UriComponentsBuilder
                .fromUriString("/restful/foos")
                .toUriString();

        Mono<String> monoResponse = rest.post(path, request);
        String response = monoResponse.block();

        if (response == null) {
            throw new IllegalArgumentException(
                    "Cannot register foo, fooRequest = " + request);
        }

        return response;
    }

    public List<Foo> findFoos() {
        //
        String path = UriComponentsBuilder
                .fromUriString("/restful/foos")
                .toUriString();

        Mono<List<Foo>> monoResponse = rest.getList(path, Foo.class);
        List<Foo> response = monoResponse.block();

        if (response == null) {
            throw new IllegalArgumentException("Cannot find foos");
        }

        return response;
    }

    public Foo findFoo(String id) {
        //
        String path = UriComponentsBuilder
                .fromUriString("/restful/foos/{id}")
                .buildAndExpand(id)
                .toUriString();

        Mono<Foo> monoResponse = rest.get(path, Foo.class);
        Foo response = monoResponse.block();

        if (response == null) {
            throw new IllegalArgumentException("Cannot find foo with id = " + id);
        }

        return response;
    }
}
