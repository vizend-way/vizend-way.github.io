Overviewâ€‹

The Lingo service in the Vizend platform handles multilingual label management across applications using the @vizendjs/vizend-layout package.

To use Lingo, your application should follow this flow:

  • Manage Labels

  • Set User Language

  • Fetch and Store Labels in Local Storage

  • Use Accent Util for Label Transformation

2. User Language Setupâ€‹

Set the user's language using LocalStorage on app load:


useEffect(() => {

  const language = localStorage.getItem("vizend.language");

  if (!language) {

    localStorage.setItem("vizend.language", "english");

  }

}, []);

Update language on user change:


const changeLanguage = (language: string) => {

  localStorage.setItem('vizend.language', language);

  window.location.reload();

};

ğŸ“˜ Be sure to force a page refresh, as language changes in localStorage are not detected automatically.

Use initializeI18nResource() from @vizendjs/lingo-view in your shared layout:
